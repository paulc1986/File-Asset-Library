# Working Files

This folder is a shareable open-source bundle for `GTA 000 - Custom Asset Library`.

Suggested GitHub contents:

- `addon/` - the Blender add-on source package
- `addon_source/` - a clean source-only copy without Python cache files
- `docs/` - quick start and user guide
- `examples/` - example metadata and library layout
- `wiki/` - GitHub wiki-ready pages
- `release/` - packaged add-on zip

Before publishing publicly, choose and add a license that matches how you want the project shared.
