# GTA 000 - Custom Asset Library

Welcome to the wiki for `GTA 000 - Custom Asset Library`.

This add-on lets Blender treat whole `.blend` files, collections, or detected object hierarchies as package-based reusable assets.

- Wiki URL: [Blender File Asset Library Wiki](https://github.com/paulc1986/Blender-File-Asset-Library/wiki)

## Core Features

- whole `.blend` packages as asset entries
- collection, hierarchy, or metadata-driven import targets
- append and link workflows
- category, sub-category, and tag libraries
- per-asset library overrides
- preview thumbnails and validation
- automatic library refresh checks at a configurable interval

## Important Notes

- Native Blender Asset Browser drag-and-drop is not available for this custom sidebar browser.
- Importing from the currently open `.blend` into itself is not supported by Blender.
- If no library roots are enabled, the browser starts empty and clears stale cached entries.
