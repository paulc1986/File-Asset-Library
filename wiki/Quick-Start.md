# Quick Start

## Install

1. Open Blender.
2. Go to `Edit > Preferences > Add-ons`.
3. Click `Install...`.
4. Select the add-on zip.
5. Enable `GTA 000 - Custom Asset Library`.

The browser appears in the `Custom Asset Library` sidebar tab in the 3D View.

## Add A Library Root

1. Open `Libraries`.
2. Open `Asset Library Paths`.
3. Click `+`.
4. Choose the folder containing your asset `.blend` files.
5. Click refresh once for the initial scan.

If `Auto Refresh Libraries` is enabled, new or changed files are picked up automatically after that.

## Browse And Import

Use the main panel to:

- search
- filter by category, subcategory, or tags
- switch between grid and list
- sort assets
- append or link packages

Use `Asset Details` to:

- switch the active asset from the filtered list
- rename an asset for your library
- assign categories, sub-categories, and tags
- validate or regenerate thumbnails
